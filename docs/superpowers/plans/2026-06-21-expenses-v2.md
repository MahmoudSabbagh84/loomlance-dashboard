# Expenses v2 (F12c) — Spec + Plan

> Origin: QA **F12c**. Mirror Time v2 for expenses: bill **per project**, and **append** to a project's existing draft invoice instead of duplicating — unifying time + expenses onto one per-project draft. Build task-by-task; lint/test/build green before each commit (scope `expenses`).

## Decisions (locked, brainstormed 2026-06-21)
- **Per-project billing** with **find-or-append**: generating for a project (from time OR expenses) appends to that project's existing **draft** invoice (matching currency) if one exists, else creates it → **one draft per project** accumulating both time and expense lines.
- **Client-only expenses** (no project): bill per **client** into a per-client draft (project_id null), same find-or-append rule.
- **Single currency per invoice:** a draft is matched/created per `(client, project-or-null, currency)`. Expenses in a different currency than an existing draft get their own draft. (Multi-currency mixing stays out of scope.)
- **UI:** an Expenses **Ready-to-bill panel** on `/expenses` (mirrors Time), with per-project rows + per-client rows; one-click generate → opens the (possibly appended) draft. Removes the old pick-a-client `GenerateExpenseInvoiceModal`.

## Data / RPCs (Task 1 — one migration, apply via MCP + file)
- **Internal `_find_or_create_draft(p_user, p_client, p_project, p_currency) returns uuid`** (SECURITY DEFINER, revoked from public/anon/authenticated): returns the latest matching `status='draft'` invoice for `(user, client, project IS NOT DISTINCT FROM p_project, currency)`, else inserts a new draft (`next_invoice_number`, dates today/+30) and returns it.
- **Modify `generate_invoice_from_time_for_project(p_project_id)`** to use `_find_or_create_draft` (was: always insert). Append line items with `position = (select coalesce(max(position),-1) from invoice_line_items where invoice_id=v_inv) + row_number() over(...)`. Currency = profile default. Keep contract→rate grouping; keep stamping + `NO_UNBILLED_TIME`.
- **New `generate_invoice_from_expenses_for_project(p_project_id, p_currency)`** (SECURITY DEFINER, granted authenticated): verify project owned by `auth.uid()`; find-or-create draft for (client, project, currency); append one line per expense (qty 1, unit = amount, desc = trimmed description or category) for billable/unbilled expenses with `project_id=p_project_id and currency=p_currency`; stamp `invoiced_on_invoice_id`; raise `NO_BILLABLE_EXPENSES` when none.
- **New `generate_invoice_from_expenses_for_client(p_client_id, p_currency)`** (same, for `project_id is null and client_id=p_client_id and currency=p_currency`); draft matched with project null.
- **Drop** old `generate_invoice_from_expenses(p_client_id)`.
